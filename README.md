# PharmaTrack - Pharmacy Inventory & Sales Management System

PharmaTrack is a full-stack MERN application designed to simplify pharmacy inventory management, purchase orders, medicine sales, vendor management, and user administration.

The system supports **Role-Based Authentication**, allowing different access levels for **Admin** and **Pharmacist** users.
